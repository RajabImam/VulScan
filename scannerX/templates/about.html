About the Team

Team of 5 student's from ISEP build VulScan as a project for a conference to submit a publication on web application vulnerabilities. Our aim and goals is to use the knowledge acquired in software security and cyber security during our time at ISEP to be able to develop a tool that can be used to detect vulnerabilities in web applications.
